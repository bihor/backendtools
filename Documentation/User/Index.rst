Users Manual
============

Just install the extension. Click on the left side at “Extension & file tools”. Select one of the 5 modes (see top left).

Note: you need admin rights! The link appears under the “Admin tools”.

.. important::

   If you are using TYPO3 9, then you will need the extension typo3db_legacy too!
